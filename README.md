# Calculation Of Wells

Приложение для импорта данных по скважинам из CSV-файла, проверки корректности данных и формирования сводной информации по каждой скважине.
Проект реализован в виде WPF/MVVM приложения на платформе .NET 8.

---

# Функциональность

* Импорт данных из CSV-файла
* Валидация строк данных
* Группировка интервалов по скважинам
* Расчёт сводной информации
* Отображение результатов и списка ошибок

---

## Формат входного файла

CSV формат (разделитель `;`):

```
WellId;X;Y;DepthFrom;DepthTo;Rock;Porosity
```

Пример:

```
A-001;82.10;55.20;0;10;Sandstone;0.18
A-001;82.10;55.20;10;25;Limestone;0.07
A-002;90.00;60.00;0;15;Shale;0.04
A-002;90.00;60.00;15;40;Sandstone;0.22
A-003;100.10;72.50;0;5;Sandstone;0.19
```

---

# Валидация

Выполняются следующие проверки:

* DepthFrom < DepthTo
* DepthFrom >= 0
* Porosity находится в диапазоне [0..1]
* Rock не должен быть пустым
* Интервалы одной скважины не должны пересекаться

Ошибки содержат:

* номер строки
* WellId
* описание ошибки

Пример:

```
Row: 5 | Well: A-001 | Error: DepthFrom must be less than DepthTo
```

---

# Расчёты

Для каждой скважины вычисляются:

* Общая глубина (максимальный DepthTo)
* Количество интервалов
* Средняя пористость (взвешенная по толщине интервалов)
* Самая распространённая порода (по суммарной толщине)

Формула средней пористости:

```
AveragePorosity =
Σ (IntervalLength × Porosity)
--------------------------------
Σ IntervalLength
```

# Использование

1. Выбрать CSV файл
2. Нажать кнопку загрузки
3. Получить результат
4. При необходимости можно очистить имеющиеся значения кнопкой "Очистка"
